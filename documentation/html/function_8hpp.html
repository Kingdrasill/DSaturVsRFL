void compareAlgorithmsForGraphs ( )

Compara os algoritmos DSatur e RLF para grafos aleatórios.

Este método é usado para comparar os algoritmos DSatur e RLF para grafos aleatórios. Para isto acontecer, primeiro é se criado um arquivo em que os grafos gerados e as soluções dos algoritmos seram guardados. Depois disto é se perguntado o números de grafos que devem ser gerados, depois o número de vértices e por último o número de arestas para os grafos a serem gerados. Com tudo setado os grafos são gerados, sendo que toda vez que um é gerado ele é colocado no arquivo, resolvido pelos dois algoritmos sendo que o tempo e solução de cada um é colocado no arquivo. Depois de todos os grafos forem gerados e resolvidos, no final do arquivo é se colocado o tempo médio de cada algoritmo.

void compareAlgorithmsForSudokus ( )

Compara os algoritmos DSatur e RLF para sudokus 2x2 ou 3x3 aleatórios.

Este método é usado para comparar os algoritmos DSatur e RLF para sudokus 2x2 ou 3x3 aleatórios. Para isto acontecer, primeiro é se criacado um arquivo em que os sudokus gerados e as soluções dos algoritmos seram guardados. Depois disto é se perguntado o tamanho do sudoku a ser gerado, depois o números de devem ser gerados, depois o número de sudokus a serem gerados e por último o número de células já devem possuir cores. Com tudo setado os sudokus são gerados, sendo que toda vez que um é gerado ele é colocado no arquivo, resolvido pelos dois algoritmos sendo que o tempo e solução de cada um é colocado no arquivo. Depois de todos os sudokus e resolvidos, no final do arquivo é se colocado o tempo médio de cada algoritmo e porcentagem de sudokus que cada algoritmo conseguiu resolver corretamente.
